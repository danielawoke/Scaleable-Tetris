With this project, I really ended up pushing the boundries of what java script is normally used for. I honestly would not recomend doing something like this, and attempt to recreate fully fledged object oriented program with java script. It's extremly difficult for what its worth, and i dont think the glory you'd get from it would be very much. However, one thing I should mention though is when it comes to big projects, java script is fast as F*$&@! No matter how many for loops i nested the program still manage to work like as if nothing happened. I hope you enjoy my project, the scaleability feauture is a little bit rusty. The over all design may not be very appealing to some, but I left it this way so that it'd replicate the feeling of early New Grounds, and the internet during the late 2000s as a way of giving tribute to that era; an era that frankly raised me.
